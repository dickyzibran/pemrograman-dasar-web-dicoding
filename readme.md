# Nobara Kugisaki Website

Website statis bertema anime yang membahas karakter **Nobara Kugisaki** dari serial *Jujutsu Kaisen*. Website ini dibuat menggunakan **HTML** dan **CSS** dengan tampilan dark mode neon aesthetic yang modern serta responsive di berbagai ukuran layar.

Pada website ini terdapat beberapa informasi mengenai Nobara Kugisaki seperti profil, kemampuan, kepribadian, peran dalam cerita, serta fakta menarik tentang karakternya. Selain itu, terdapat sidebar tambahan yang menampilkan informasi singkat mengenai anime *Jujutsu Kaisen*.

Project ini dibuat sebagai latihan dasar pengembangan web dengan menerapkan:

- Semantic HTML
- Flexbox layout
- Responsive design
- Media query
- Custom font
- Hover animation
- Sticky navigation bar
- Card design dan efek neon glow
